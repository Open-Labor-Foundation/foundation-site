# Contributing to foundation-site

This repository contains the source for [openlabor.foundation](https://openlabor.foundation).

## What belongs here

- Content updates to `content/`
- Theme changes under `themes/olf/`
- Hugo configuration in `hugo.toml`

Do not commit the `public/` build output — it is gitignored and generated by Cloudflare Pages at deploy time.

## How to contribute

1. Fork the repository
2. Make your changes and preview locally with `hugo server`
3. Open a pull request with a brief description of what changed and why

## Code of conduct

See [CODE_OF_CONDUCT.md](CODE_OF_CONDUCT.md). The full OLF code of conduct is in [open-labor-foundation](https://github.com/Open-Labor-Foundation/open-labor-foundation/blob/main/CODE_OF_CONDUCT.md).
