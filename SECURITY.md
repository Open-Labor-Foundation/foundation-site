# Security Policy

This repository contains only static site source. There is no server, no database, and no application runtime.

For security concerns related to the OLF platform or software repositories, see the full policy in [open-labor-foundation](https://github.com/Open-Labor-Foundation/open-labor-foundation/blob/main/SECURITY.md).

Report vulnerabilities to **[security@openlabor.foundation](mailto:security@openlabor.foundation)**.
